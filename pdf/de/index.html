Häufig gestellte Fragen zur PDF-Komprimierung

Wie funktioniert die PDF-Komprimierung?

Unser Tool rendert jede PDF-Seite als Bild und kodiert sie als JPEG in der gewählten Qualität neu. Das Ergebnis ist eine leichtere PDF.

Wie stark kann eine PDF-Datei reduziert werden?

Das hängt vom Inhalt ab. PDFs mit vielen hochauflösenden Bildern können um 70-90% reduziert werden.

Kann ich Text in der komprimierten PDF auswählen?

Da Seiten als Bilder gerendert werden, ist der Text nicht auswählbar. Für auswählbaren Text verwenden Sie die höchste Qualitätsstufe.

Gibt es eine Dateigrößen- oder Seitenbeschränkung?

Es gibt keine von uns auferlegte Grenze. Die eigentliche Grenze ist der Arbeitsspeicher Ihres Browsers.

Sind meine PDFs privat?

Absolut. Ihre Dateien werden nie an einen Server gesendet. Die gesamte Verarbeitung erfolgt lokal auf Ihrem Gerät.
